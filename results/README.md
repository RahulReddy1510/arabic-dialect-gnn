# results/

This directory is generated by `evaluation/evaluate.py` and `evaluation/error_analysis.py`.
All files here are **outputs**, not inputs — they are `.gitignore`d by default (see `.gitignore`).

## Files

| File | Generated by | Contents |
|---|---|---|
| `full_results.json` | `evaluation/evaluate.py` | Complete nested results for both models: per-dialect F1, confusion matrix, bootstrap CI, per-length F1 |
| `full_results.csv` | `evaluation/evaluate.py` | Flat summary CSV — this is the file used for tables in the README |
| `eval_results.csv` | `evaluation/evaluate.py` | Simplified 2-row comparison table |
| `class_distribution.png` | `notebooks/01_data_exploration.ipynb` | Bar chart of dialect class counts |
| `phoneme_graphs.png` | `notebooks/02_graph_construction.ipynb` | Networkx graph visualizations, one per dialect |
| `per_dialect_f1_comparison.png` | `notebooks/03_model_comparison.ipynb` / `evaluation/error_analysis.py` | Grouped bar chart |
| `macro_f1_comparison.png` | `notebooks/03_model_comparison.ipynb` | Error-bar plot with 95% bootstrap CI |
| `ablation_study.png` | `notebooks/03_model_comparison.ipynb` | Horizontal bar chart of ablation results |
| `confusion_matrices_comparison.png` | `evaluation/error_analysis.py` | Side-by-side heatmaps |
| `attention_visualization.png` | `notebooks/04_error_analysis.ipynb` | Per-phoneme attention heatmap |
| `per_length_f1.png` | `notebooks/04_error_analysis.ipynb` | F1 vs utterance length line plot |
| `confused_pairs.png` | `notebooks/04_error_analysis.ipynb` | Confused dialect pair bar chart |
| `gat_confusion_matrix.png` | `evaluation/error_analysis.py` | GAT confusion matrix (standalone) |
| `arabert_confusion_matrix.png` | `evaluation/error_analysis.py` | AraBERT confusion matrix (standalone) |

## Reproducing results

```bash
# 1. Train GAT
bash scripts/run_training.sh

# 2. Train AraBERT baseline
bash scripts/run_baseline.sh

# 3. Run full evaluation
bash scripts/run_evaluation.sh

# 4. Run notebooks (optional, for plots)
jupyter nbconvert --to notebook --execute notebooks/03_model_comparison.ipynb
jupyter nbconvert --to notebook --execute notebooks/04_error_analysis.ipynb
```

## Reported numbers (from full_results.csv)

| Model | Macro F1 | Gulf | Egyptian | Levantine | Maghrebi | Iraqi | 95% CI |
|---|---|---|---|---|---|---|---|
| AraBERT | 81.7% | 79.3% | 84.1% | 82.4% | 80.2% | 78.9% | [80.8, 82.5] |
| GAT (ours) | **84.2%** | **82.1%** | **85.9%** | **84.7%** | **83.8%** | **84.5%** | [83.4, 85.0] |

The GAT outperforms AraBERT by **+2.5 macro F1** points. The biggest gains are on Iraqi (+5.6) and Maghrebi (+3.6), the dialects where AraBERT's BPE tokenization fragments the most phonologically distinctive vocabulary.
